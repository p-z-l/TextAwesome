**A lightweight text editor for iOS**

## Features
- Creat/View/Edit multiple types of documents
> txt,swift,python,html,css,js,etc.

- iOS Native document browser
- Dark Mode support
- Customized font style & font size
- Text searching
- Basic code highlight for Swift
- iPadOS cursor support
- iPadOS keyboard shortcuts support
